# 🎙️ Roteiro Estratégico de Entrevista (Método STAR)

Este documento contém o pitch de 2 minutos estruturado para apresentar o projeto ChurnGuard AI para recrutadores e gestores técnicos em entrevistas de emprego.

---

## 🗣️ O Pitch de 2 Minutos

"Uma das minhas principais realizações técnicas recentes foi o desenvolvimento do **ChurnGuard AI**, um ecossistema inteligente de ponta a ponta projetado para sanar uma das maiores dores de receita das empresas atuais: o cancelamento de clientes. 

* **[SITUAÇÃO]** Eu identifiquei que muitas empresas perdem faturamento de forma silenciosa porque atuam apenas após o cliente cancelar, quando o custo de reaquisição já é até 7 vezes maior do que o de retenção.
* **[TAREFA]** Minha meta foi construir um sistema preventivo e modular que unisse Engenharia, Ciência de Dados e Desenvolvimento de Software para antecipar o risco desses clientes e calcular o impacto financeiro em tempo real.
* **[AÇÃO]** Para isso, eu desenvolvi uma arquitetura em quatro camadas. Primeiro, criei uma esteira de **Engenharia de Dados (ETL)** em Python e Pandas que trata os dados desde a camada bruta à camada analítica. Na sequência, na frente de **Data Science**, treinei e validei um modelo preditivo baseado em *Random Forest* que atingiu **94% de acurácia** na probabilidade de churn. 

Pensando em produção, desenvolvi uma **API assíncrona robusta via FastAPI** e Pydantic para receber dados cadastrais e retornar scores de risco em milissegundos. Por fim, para traduzir o código em valor de negócio, construí um **Dashboard executivo no Streamlit**, que exibe métricas financeiras críticas como LTV, receita recorrente mensal (MRR) e gera alertas automáticos para o time de retenção agir a tempo. Todo o ecossistema foi empacotado em **Docker Containers** para simular um deploy real em nuvem.

* **[RESULTADO]** Como resultado, o projeto demonstrou como utilizar a inteligência artificial não apenas para gerar relatórios passivos, mas como uma ferramenta ativa de tomada de decisão capaz de blindar o caixa da empresa e mitigar perdas de receita."

---

## 🎯 Pilares Importantes para Destacar
1. **Foco no ROI:** O discurso começa e termina abordando receita e faturamento.
2. **Domínio Técnico:** Acelera a validação ao citar explicitamente o fluxo (ETL -> Modelagem -> API -> Dashboard -> Docker).
3. **Métricas Precisas:** Menção firme aos 94% de acurácia obtidos no treino.
